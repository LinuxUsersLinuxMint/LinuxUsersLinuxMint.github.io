What is AdWM?

AdWM (Advanced Windows Manager) is an advanced management application for Windows developed by LinuxUsersLinuxMint.


Why AdWM?

AdWM software has features such as "Quick cleaning" and "System repair" for the Windows operating system, and you can perform them all with one click, and AdWM provides a simple management for windows.


Why will AdWM be rewritten?

As LinuxUsersLinuxMint, we think that AdWM is inadequate in some areas, the most basic issue is that AdWM does not have a Graphical User Interface (GUI), so we decided to rebuild AdWM to eliminate this deficiency.


What are the innovations and differences in the new version of AdWM?

One of the innovations in the new version of AdWM is that AdWM will now have a Graphical User Interface (GUI) and AdWM will now be able to perform everything more simply.
