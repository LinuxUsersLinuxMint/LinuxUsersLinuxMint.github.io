- LinuxUsersLinuxMint philosophy in brief -

As LinuxUsersLinuxMint, we aim to offer the best and most comfortable software to the user and we do our work accordingly. As LinuxUsersLinuxMint, our priority is to make our software accessible. We pay attention to accessibility in our software and we pay attention to the usability and sustainability of our software. In this article, we will try to explain our thoughts to you in the best way possible.


- LinuxUsersWhat does the LinuxMint philosophy aim for? -

LinuxUsersLinuxMint philosophy aims for better, accessible, maintainable and ease of use, which is why we have "Help / About" sections in all our software.


- Our principles as LinuxUsersLinuxMint -

1. User satisfaction

As LinuxUsersLinuxMint, we care about user satisfaction in our software, so we publish our software as open source so that users can let us know their opinions and problems/deficiencies in our software, and we organize the file structure of our software before presenting it so that the user can use the software comfortably.


2. Availability of software

LinuxUsersLinuxMint attaches importance to the usability of all our software, so we organize our software as best as possible (before it is made available for use). As LinuxUsersLinuxMint, we also try to contribute to projects (software) made by other people. (Example: "Artado Search")


3. Software that is (at least) as hassle-free as possible

As LinuxUsersLinuxMint, we organize and optimize our software before presenting it (we try to do the operations in the most comfortable/simplest ways) and test our software before presenting it. If we have not tested our software, we state this. (Ã–rn: "PyTerm BETA") Because we want users to trust us and use our software as comfortably as possible.






4. Honesty (Reliability)

As LinuxUsersLinuxMint, we take care to be honest and trustworthy, we do not deceive the user commercially in any way. Reliability is one of the most important principles for us and we want users to trust us and use all our software without any doubt.


5. Ease of use

As LinuxUsersLinuxMint, we ensure that all the software we produce is as easy to use as possible because we believe that users should be able to use a software as smoothly and comfortably as possible.
